public static void generate(string class, string stub_class, array methods)

Clones a class' interface and creates a stub version that can have return values set.

Parameters:
class - Class to clone.
stub_class - New class name. Default is the old name with "Stub" prepended.
methods - Additional methods to add beyond those in the cloned class. Use this to emulate the dynamic addition of methods in the cloned class or when the class hasn't been written yet.
