Class ClassDoc

public class ClassDoc
extends ProgramElementDoc

Represents a PHP class and provides access to information about the class, the class' comment and tags, and the members of the class. A classDoc only exists if it was processed in this run of PHPDoctor. References to classes which may or may not have been processed in this run are referred to using type (which can be converted to classDoc, if possible).
