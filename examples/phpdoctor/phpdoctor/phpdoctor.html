Class PHPDoctor

PHPDoctor

public class PHPDoctor

This class holds the information from one run of PHPDoctor. Particularly the packages, classes and options specified by the user. It is the root of the parsed tokens and is passed to the doclet to be formatted into output.

createTag

public Tag createTag(str name, str text, str data, RootDoc root)

Create a tag. This method first tries to load a Taglet for the given tag name, upon failing it then tries to load a PHPDoctor specialised tag class (e.g. classes/paramtag.php), if it still has not found a tag class it uses the standard tag class.

Parameters:
name - The name of the tag
text - The contents of the tag
data - Reference to doc comment data array
root - The root object
